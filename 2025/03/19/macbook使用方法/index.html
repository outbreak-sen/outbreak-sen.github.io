macbook使用方法

发表于2025-03-19|更新于2025-03-19
|浏览量:

Macbook使用笔记

  • 左右分屏：按住Option键拖移窗口，或将指针放在左上角的绿色图标上
  • 分屏幕：三指往上推然后再在顶部把两个软件单指结合到一起
  • delete键按：fn+删除键，可以删除光标后面的字符
  • 直接到达桌面：四个指头放大然后手拿开
  • 切换输入法：control空格
  • 删除文件：command+delete
  • Safari新建标签页打开：command+左键
  • 截图大全
    1. 全屏截图并保存到剪切板：Control + Command + Shift + 3
    2. 选择区域截图并保存到剪切板：按下Control + Command + Shift + 4
    3. 全屏截图：com➕shift➕3（ipad标注同步）
    4. 矩形截图：com➕shift➕4 （空格移动，option中心缩放）
    5. 录屏工具！！！！！：com➕shift➕5
  • 退出应用直接退出：com➕Q
  • 隐藏窗口：com➕H
  • 关闭窗口：com➕W
  • 新建窗口：com➕N
  • 最大化/最小化窗口：com+ctrl+F
  • Mac的鼠标滚轮是反的，但是改回去触摸板又变成反的了，别扭很正常
  • ¼音量：option➕shift➕增/减音量
  • Safari激活地址栏 com➕L
  • 应用栏：程序坞｜最近使用｜缩小的和
  • 聚焦搜索：command+空格打开spotlight
  • 顶部右边的菜单图标，command拖动可以更换位置，也可以拖出来删除
  • 打开应用：四个指头缩小然后手拿开
  • 安装软件：DMG文件相当于ISO文件磁盘，之后还需退出磁盘，
  • 卸载软件：访达中应用程序把他扔到垃圾桶就行了
  • 切换窗口应用com➕tab，回退com➕shift➕tab，退出esc
  • 复制多一份文件：com➕D（创建多一份副本）
  • 移动（非复制）文件 Ctrl➕C ，com➕option➕V / com➕拖拽

文件管理

最近使用包含了以下两个部分icloud和本机中最近打开的文件
有两个部分

  • iCloud：全生态共享，包含了当前mac桌面，有些 app 会有自己的 iCloud 文件夹，既解决了备份同步，又自动创建了分类文件夹。比如 MindNode 会创建自己的 iCloud 文件夹
  • 本机： 可以显示本机，但是如果要快速浏览则包含以下两个部分
    • 下载
    • 应用程序:想把下载的软件安装进应用页面，直接拖到应用程序文件夹中
      美区id：https://appid.mynatapp.cc/wx/index/aget/9b482d5e-2ad5-44e4-ac7f-c055df9b7566

在终端中进入cloud目录方法： cd Library/Mobile\ Documents/com~apple~CloudDocs
